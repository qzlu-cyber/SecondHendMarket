## 运行

### 先配置科大讯飞 api 秘钥

[讯飞开放平台](https://www.xfyun.cn/services/voicedictation?ch=bdtg&renqun_youhua=646957)上可以注册使用，注册成功并创建一个应用后即可获得

![](./xunfei.png)

配置完成后

```
 npm install
 npm start
```

## 文件说明

- aduio: 保存讯飞音频
- srt: 保存生成的字幕
- wyyVideo: 保存最后合成的视频文件
